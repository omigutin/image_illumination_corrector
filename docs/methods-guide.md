# Methods Guide

Этот файл кратко объясняет, какие методы есть в проекте и как они влияют на результат.

## Что делает утилита

Утилита помогает:

- убрать затемнение по краям;
- выровнять яркость по всему кадру;
- подобрать параметры вручную;
- сохранить удачные настройки и повторить их на серии файлов.

## Основная идея

Обработка идёт в три шага:

1. оценивается плавный фон;
2. по нему исправляется кадр;
3. при необходимости растягивается контраст.

## Методы оценки фона

### Gaussian blur

Подходит, когда затемнение плавное и мягкое.

Главные параметры:

- `Ядро фона`
- `Sigma фона`
- `Ядро предразмытия`

### MorphologyEx

Подходит, когда обычное размытие плохо отделяет фон от деталей.

Главные параметры:

- `Операция`
- `Форма ядра`
- `Размер ядра`

### Flat-field по опорному кадру

Подходит, когда есть отдельный ровный эталонный кадр.

Главные параметры:

- `Опорный кадр`
- `Ядро сглаживания опорного кадра`

### Радиальная модель

Подходит, когда затемнение похоже на круг или овал от центра к краям.

Главные параметры:

- `Степень полинома`
- `Шаг выборки`
- `Смещение центра по X/Y`

## Методы коррекции

### Мультипликативная коррекция

Основной и самый практичный метод для виньетки.

### Вычитание фона

Спокойный вариант, если фон ведёт себя как добавочная тень.

### Гомоморфная коррекция

Мягкий вариант для более деликатной работы с яркостью.

### Только CLAHE

Локальное усиление контраста без явной карты фона.

## Параметры результата

### Авторастяжение контраста

Полезно для визуального просмотра.

### Нижний и верхний процентили

Задают нижнюю и верхнюю границы растяжения контраста.

## Практический старт

Если не знаете, с чего начать:

1. `Gaussian blur`
2. `Мультипликативная коррекция`
3. подобрать `Ядро фона`
4. подправить `Максимальный gain`

Если есть хороший референс:

1. `Flat-field по опорному кадру`
2. `Мультипликативная коррекция`

Если виньетка похожа на ровный круг:

1. `Радиальная модель`
2. степень `2` или `3`
3. при необходимости сместить центр
